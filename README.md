# BaseKit for Bukkit

### Overview
**BaseKit** is a multi-functional base plugin for high-version Minecraft servers.  
It provides a variety of core features to help server admins and players manage the server and enhance gameplay without
installing multiple plugins.

### Features
- Convenient commands to manage players and server
- Useful tools and shortcuts
- Customizable configuration for personalized server setup
- Compatible with high Minecraft versions, stable and reliable

### Installation
1. **[Download](https://github.com/Ethanz0x0/bukkit-basekit/releases)** the latest `.jar` of the plugin.
2. Place the `.jar` file into your server's `plugins` folder.
3. Start or restart the server and configure the plugin.
4. Done!

### Common Commands
- All the available commands can be found using `/basekit help` in-game
